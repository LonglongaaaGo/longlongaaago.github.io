<b> Brief description:</b>
<div style="text-align: justify">Grouping images into different themes is a challenging task in photo book curation. Unlike image object recognition, image theme recognition focuses on the main subject or overall meaning conveyed by an image. This work studies few-shot image theme recognition with pre-trained contrastive language-image models, and proposes a text-embedding-guided adaptation framework with a text-embedding-guided classifier and an auxiliary classification loss. The framework exploits visual and text features to stabilize training and improve recognition performance. We also introduce Theme25, a newly annotated dataset for image theme recognition, and evaluate the method on Theme25, CIFAR100, and ImageNet.</div>
